~~~~# Cross Bus Explorer
Cross Bus Explorer is a Cross Platform Service Bus Explorer (Windows and MacOS).

## Features

- List, Edit, Delete Queue, Topics, Subscriptions
- Purge messages
- Requeue messages
- Send messages
